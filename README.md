# 🏠 StayNear

StayNear is a clean and responsive front-end web application designed to help students and working professionals discover suitable PGs (Paying Guest accommodations) and hostels. The website provides an intuitive interface for browsing accommodations, exploring popular categories, and registering or logging into the platform.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yogeshdudi111-spec/StayNear.git
```

### Open the project

Navigate to the project folder:

```bash
cd StayNear
```

Open `index.html` in your preferred browser.

No additional installation or dependencies are required.
